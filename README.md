# рҹҡҖ Installation Guide

# 1пёҸвғЈ Clone Repository

```bash
git clone https://github.com/Yug-Vahanka/AEGIS-DEFEND.git
cd AEGIS-DEFEND
```

---

# 2пёҸвғЈ Install Backend Dependencies

```bash
cd backend

npm install express mongodb cors dotenv
```

---

# 3пёҸвғЈ Configure Environment Variables

Create a `.env` file inside `/backend`

```env
PORT=3000
MONGO_URI=mongodb://127.0.0.1:27017/aegis_defend
```

---

# 4пёҸвғЈ Start MongoDB

## Windows

```bash
net start MongoDB
```

## Manual Start

```bash
mongod --dbpath D:\MongoDB\data\db
```

---

# 5пёҸвғЈ Run Server

```bash
node server.js
```

---

# 6пёҸвғЈ Open Application

```text
http://localhost:3000
```
